## Installation

Steps to install and set up the project:

1. Clone the repository:
   ```bash
   git clone https://github.com/KeshavDussal/MERN.git
   ```
2. Navigate to the project directory:
   cd MERN

3. Install dependencies:
   npm install

4. Start the application:
   npm start
